# Agilent 8900 ICP-MS — Complete Operation Guide

A comprehensive, chapter-by-chapter guide for operating the **Agilent 8900 Triple Quadrupole ICP-MS (ICP-QQQ)**, used for pharmaceutical heavy metal analysis at NPRA Malaysia.

## 📖 Chapters

| # | Chapter | Topics |
|---|---------|--------|
| 1 | **Introduction & System Overview** | ICP-MS fundamentals, triple quad technology, all key components, MS/MS vs SQ |
| 2 | **Startup & Shutdown** | Pre-flight checklist, power-on sequence, plasma ignition, shutdown procedures |
| 3 | **MassHunter Software & Tuning** | MassHunter interface, autotune, cell gas modes, performance reports, IntelliQuant |
| 4 | **Sample Preparation & Acid Digestion** | USP ⟨233⟩ prep methods, microwave digestion, decontamination, standard/ISTD prep, spike recovery |
| 5 | **Method Development & Data Acquisition** | Method wizard, element selection, calibration, batch setup, FullQuant, Batch-at-a-Glance |
| 6 | **Maintenance & Troubleshooting** | Daily/weekly/monthly schedule, cone cleaning, torch/nebulizer care, common issue resolution |
| 7 | **Regulatory Compliance** | ICH Q3D, USP ⟨232⟩/⟨233⟩, PDE table, element classification, validation, audit trail |
